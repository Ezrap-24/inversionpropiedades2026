# Programa Patrimonial Minero de Inversión

Landing page para el programa de inversión inmobiliaria dirigido a socios de sindicatos mineros en Chile.

## Descripción

Sitio web estático que presenta el **Fundo Puerto Octay** como oportunidad de inversión patrimonial exclusiva para trabajadores mineros. Incluye información de los proyectos Norte y Sur, ubicaciones, galería de imágenes, guías de inversión descargables y formulario de contacto.

## Estructura

```
├── index.html                  # Página principal
├── css/
│   └── styles.css
├── js/
│   └── script.js
├── pages/

│   ├── ubicacion-sur.html
│   ├── educacion-financiera.html
│   ├── politica-privacidad.html
│   └── terminos-condiciones.html
└── assets/
    ├── images/
    ├── pdfs/
    └── video/
```

## Tecnologías

- HTML5 / CSS3 / JavaScript vanilla
- Google Fonts (Hanken Grotesk)
- Deploy en Vercel

## Deploy

El sitio se despliega automáticamente en Vercel al hacer push a la rama `main`.
